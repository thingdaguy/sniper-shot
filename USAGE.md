# Usage
1. Click the extension icon (top-right toolbar) to open the popup.
2. Paste your prompt and Gemini API key, then click `Save Prompt + API Key`.
3. Press `Ctrl + Shift + K` to send prompt + page context to Gemini in the background and save the result.
4. Press `Ctrl + Shift + L` to toggle the saved result open/close at bottom-right.
5. Press `Ctrl + Shift + X` to kill/destroy the extension logic permanently.
